# Seguridad y privacidad

Los chats pueden contener **datos personales**.

## No subir a Git
- Archivos de entrada (xlsx/csv) reales
- Exports con datos reales
- `.env` (puede contener credenciales)
- Logs con información sensible

## Recomendaciones
- Trabaja con copias anonimizadas cuando sea posible.
- Controla el acceso a la carpeta `outputs/`.
- Si un secreto se sube por error, rótalo/rota inmediatamente y limpia historial del repo remoto.
