# Contribuir

## Reglas rápidas
- No subir datos, audios, transcripciones, excels con PII.
- No subir credenciales. Usar `.env` y mantenerlo en `.gitignore`.
- Antes de hacer PR/commit:
  - limpiar outputs del notebook
  - verificar que `requirements.txt` sigue vigente

## Estilo
- Mantener configuración en un bloque único.
- Preferir variables de entorno para rutas/credenciales.
- Agregar notas en `docs/DOCUMENTATION.md` si cambias el flujo.
