# Seguridad y privacidad

Los audios y transcripciones pueden contener datos personales.

## No subir a Git
- `inputs/audios/*`
- `inputs/txt/*`
- `outputs/*`
- `.env`

## Recomendaciones
- Usa muestras anonimizadas cuando puedas.
- Comparte outputs agregados (no texto completo).
- Rota credenciales si se filtran por error.
